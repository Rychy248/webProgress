The Hyper Terminal

The hyper terminal is a powerfull tool for interacting with the computer, without any graphic interface


  1. For Linux users: Terminal is preinstalled so we don't need to install anything
  2. For MACos users: Terminal is preinstalled too as the same linux users, so we don't need to install anything
  3. For Windows users: Terminal isn't preinstalled, they have shell, but ins't like the unix comnad line, so we can install a hyper terminal, or git, for using the git bash, who use the unix comand line instructions to interact with the system

SOme of the most common comands:

  1. cd: To change the directory.
  2. ls: To list the items of a directory
  3. pwd: Show the actual path
  4. history: Show the recent used commands
  5. ping: This check if an dispositive is active
  6. mkdir: Make a directory
  7. rm: Delete elements, for directories this need the option -r (revursive way)
  8. cat: Filtering the output of some comamnd
  9. shutdown: Shutdown the system
  10. ifconfig: Wireless configuration
  11. sudo: Acces to super user
  12. su: Acces to another user
  13. type:To see the type of a command
  14. Wich:Show the actual path or packege of a command
